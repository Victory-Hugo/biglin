名词解释

Single-cell analysis reveals contextdependent,cell-level selection of mtDNA
Nature Article Published online: 24 April 2024

  1. LHON (Leber's Hereditary Optic Neuropathy) - 莱伯遗传性视神经病变：一种遗传性疾病，导致中心视力急速且永久性下降。该病通常由mtDNA突变引起。
  2. SILENT DdCBE - 静默DdCBE（Directed DNA deaminases Combined with Base Editors的缩写）：一种基因编辑工具，可引入无影响蛋白编码的同义突变。
  3. SCI-LITE - SCI-LITE（Single Cell Combinatorial Indexing for Transcriptome and Epigenome的缩写）：一种用于同时捕获单细胞内转录组和表观遗传信息的高通量技术。
  4. OXPHOS (Oxidative Phosphorylation) - 氧化磷酸化：细胞呼吸过程中的一个阶段，主要在线粒体内进行，用于生成ATP。
  5. MELAS (Mitochondrial Encephalopathy, Lactic Acidosis, and Stroke-like episodes) - 线粒体脑病、乳酸中毒和类中风发作综合症：一种由mtDNA突变引起的罕见疾病，影响多个系统，特别是中枢神经系统和肌肉。
  6. MERRF (Myoclonic Epilepsy with Ragged Red Fibers) - 肌阵挛癫痫伴破碎红纤维：一种遗传性疾病，特征是突发的肌阵挛、癫痫发作和肌肉活检下观察到的破碎红纤维。
  7. mtDNA (mitochondrial DNA) - 线粒体DNA：存在于线粒体中的DNA，与核DNA不同，mtDNA遗传自母亲。
  8. DdCBE (Directed DNA deaminases Combined with Base Editors) - 定向DNA脱氨酶结合碱基编辑器：一种基因编辑技术，用于精确修改DNA序列。
  9. ONC DdCBE - 癌症相关的DdCBE：特定设计用于引入可能与癌症相关的突变的DdCBE工具。

摘要

  1. 异质性现象：本研究探讨了单个细胞中野生型和突变型线粒体DNA（mtDNA）共存的现象，即异质性。异质性水平在发育、疾病和衰老过程中会动态变化，但其具体原因尚不明确。
  2. 研究方法与技术：
    • 精确的 mtDNA 基因编辑：使用 DdCBE 技术对 mtDNA 进行精确编辑。
    • SCI-LITE技术：采用单细胞组合索引技术（SCI-LITE），通过超高通量追踪单细胞异质性。
  3. 实验设计：实验中构建了含有同义或非同义复合体I mtDNA突变的细胞，并在标准培养条件下观察这些细胞群体如何淘汰非同义突变，而同义变异则被保留。
  4. 选择与漂移：研究表明，在影响群体异质性的过程中，选择作用优于简单的遗传漂移。
  5. 细胞和细胞内水平的作用：通过同时追踪单细胞mtDNA异质性和祖系，发现尽管群体异质性发生变化，但单个细胞谱系的异质性保持稳定，表明选择作用主要影响分裂细胞的细胞适应性。
  6. 环境的影响：通过将细胞置于已报告能增强细胞适应性的环境中，实验表明可以促使细胞积累高水平的截断复合体I mtDNA异质性。这表明非同义mtDNA异质性对细胞适应性的影响取决于环境条件。

结论：在分裂细胞中，特定的非同义 mtDNA 异质性可能对细胞适应性产生有害、中性甚至有益的影响，但其效应的正负完全取决于环境。

介绍

  1. 线粒体DNA（mtDNA）的特性：mtDNA是母系遗传的，编码氧化磷酸化（OXPHOS）机制的关键组分。mtDNA的突变率高于核DNA，异质性发生在突变型mtDNA和野生型共存的情况下。

  2. mtDNA突变的影响：

    • 遗传mtDNA突变：可导致严重的OXPHOS疾病。
    • 体细胞mtDNA突变：在多种癌症中观察到，少数情况下可驱动肿瘤生成。
    • 随着年龄增长的异质性积累：与退行性疾病有关。

  3. 异质性动态的复杂性：

    • 在细胞培养和体内环境中极其复杂。
    • 在母系传播的疾病中，异质性水平在同一个体的不同组织中存在差异，甚至在兄弟姐妹之间也有显著差异。

  4. 异质性动态的研究方法：

    • 使用依赖于线粒体OXPHOS的可以在细胞培养中通过“异质性转移”减少突变型mtDNA。
    • 研究了异质性转移是由漂移还是选择驱动的不确定性，如果是后者，不清楚它是在细胞适应性还是细胞内（如mtDNA周转和线粒体自噬）水平上操作。

  5. 单细胞异质性研究的重要性与挑战：

    • 大多数研究仍然使用携带自然发生的mtDNA突变的细胞的大量群体进行，因为如CRISPR这样的工具无法作用于mtDNA。
    • 单细胞异质性分析具有重要性，但实现起来非常困难。

  6. 新技术的应用：

    • 结合了最近引入的mtDNA基础编辑技术和SCI-LITE，后者是一种高度可扩展、敏感、成本有效且灵活的单细胞特定转录测量工具。
    • 在单细胞水平上，工程化并监测了同义和非同义变异的异质性动态，跨越细胞群体和克隆谱系。

SCI-LITE技术用于单细胞mtDNA分析

  1. 现有技术的局限性：

    • 多种单细胞分析技术已被描述，包括基于微孔板和基于微滴的方法。然而，这些方法都不适合大规模单细胞mtDNA异质性分析。基于微孔板的方法需要大量手动操作，而基于微滴的方法成本过高，需要专门的微流控设备。

  2. SCI-LITE技术介绍：

    • SCI-LITE利用单细胞组合索引（SCI）技术，通过生成组合分子标签，从而概率性地识别单个细胞。
    • SCI-LITE调整了sci-RNA测序（sci-RNA-seq）和Split-seq的组合索引方法，提高了可扩展性并降低了测序成本。

  3. 条形码策略和数据分析：

    • 每个转录本根据其四种条形码的组合被分配到特定的细胞。
    • 为分析SCI-LITE数据开发了免费可用的软件包。

  4. mtDNA 和 mtRNA 丰度的检测：

    • 利用SCI-LITE研究乙啶溴（EtBr）处理对mtDNA和mtRNA丰度的影响，EtBr抑制mtDNA的复制和转录。
    • 发现在EtBr处理下，整体mtRNA水平低于mtDNA水平，SCI-LITE能精确反映单细胞中的mtRNA水平。

编辑引入的mtDNA异质性分布的复杂性和选择压力对异质性动态的影响

  1. 实验设计：

    • 在之前研究中引入的DdCBE技术（特定于MT-ND4基因的突变）被用来转染293T细胞，并在不同时间点（10小时、24小时、72小时后）使用SCI-LITE进行分析。

  2. 异质性分布的观察：

    • 初步的整体分析显示的异质性与之前报道相符，但SCI-LITE揭示了明显的双峰异质性分布，表明编辑过的细胞从低异质性快速过渡到高异质性。

  3. 编辑效率与细胞周期的关系：

    • 发现一些细胞在一个细胞分裂周期内的异质性远超50%，这可能是由于在一个细胞周期内多轮mtDNA复制。

  4. 表达与活性的关系：

    • 通过使用新开发的带有荧光标记的编辑器进行实验，发现异质性水平与DdCBE表达水平密切相关。

  5. 非同义与同义突变的选择压力：

    • 通过在含糖的培养基中培养经过非同义和同义突变编辑的细胞，观察到非同义变异在含半乳糖培养基中受到强烈选择压力，而同义变异则未见明显选择。
    • 这种结果表明系统中的异质性转移不是由简单的漂移造成的，而是受到选择的影响。

错义异质性的细胞显示出较低的适应性和生长速率

  1. 选择作用的层级：

    • 研究旨在确定明显的选择作用是在细胞水平还是细胞内水平发生。通过转染LHON或SILENT编辑器，并根据DdCBE表达高低进行细胞分选，随后测量它们在培养中的适应性。

  2. 适应性观察：

    • 在低异质性水平的LHON编辑或SILENT编辑细胞中，没有观察到任何适应性缺陷。
    • 然而，在高异质性的细胞群中，LHON编辑的细胞生长速度慢于SILENT编辑的细胞，显示出LHON编辑细胞群的适应性缺陷。

  3. 培养条件的影响：

    • 在含半乳糖的培养基中，LHON编辑细胞显示出明显的生长缺陷和大量细胞死亡，而SILENT编辑细胞则未见这些现象。

  4. 复杂I缺陷的验证：

    • 比较了LHON编辑和SILENT编辑细胞的呼吸速率，发现LHON编辑细胞的呼吸速率低于SILENT编辑细胞，与MT-ND4错义突变引起的复杂I缺陷一致。

  5. 异质性水平的动态跟踪：

    • 使用SCI-LITE跟踪多个等位基因在单个细胞内的共存情况。在培养5天、10天和15天后，观察到处理过LHON DdCBE的细胞中错义异质性减少，而静默异质性增加。

  6. 异质性阈值模型的应用：

    • 提出了一个基于“异质性阈值”的模型，该模型假设当异质性低于某个水平（通常为60-90%）时，可能不会产生强烈的表型。到了第10天，大多数细胞的错义异质性已低于60%。

  7. 随机漂移与选择作用的区分：

    • 使用基于“Kimura”分布的简单漂移模型分析了观察到的静默异质性分布，与静默异质性的实际数据拟合良好，而错义异质性的拟合较差，表明错义异质性的变化不能仅由简单随机漂移解释。
这部分研究通过实验证实了当错义mtDNA异质性水平降低至特定阈值以下时，细胞可能不会展现出与高异质性相关的病理性表型，支持了异质性阈值模型的合理性和重要性。

mtDNA异质性对细胞适应性

  1. 多路复用的SCI-LITE实验：这是一种高通量的单细胞分析技术，能够同时监测单个细胞中的异质性和细胞谱系。这种技术允许研究者在单个细胞级别上观察基因表达和遗传变异。

  2. 使用LHON或SILENT DdCBE转染细胞：这里使用的是两种特定的基因编辑工具，分别针对带有疾病相关（LHON）和非病理性（静默，SILENT）的mtDNA突变。

  3. 慢病毒库转导编辑细胞：通过慢病毒转导方法将具有唯一谱系条形码的库引入已编辑的细胞。这些条形码帮助追踪细胞的谱系，即细胞的“家族历史”.

  4. 错义 mtDNA 变异的纯化：如果细胞水平的选择作用导致突变的 mtDNA（错义突变）被清除，那么理论上，具有高错义异质性的细胞谱系会因生长缺陷而在实验后期消失。

  5. 异质性水平稳定：在单个细胞谱系中，不同时间点测量的异质性水平相对稳定，表明在细胞内部并没有强烈的选择压力改变这些水平。

  6. 高错义异质性谱系的消失：在实验的早期和晚期时间点之间，高错义异质性的细胞谱系消失，这表明这些细胞由于其较差的适应性而无法生存。

  7. 缺陷OXPHOS的适应性优势：在特定环境下（如低氧），OXPHOS功能受损的细胞可能显示出适应性优势。这种情况在特定类型的癌症（如Hürthle细胞癌）中尤为明显，其中复合体I的突变可能有助于癌细胞的生长和存活。

  8. 体内实验：将经过错义mtDNA突变编辑的细胞植入到免疫缺陷小鼠中，直接观察这些突变是否促进了肿瘤的生长，以此验证这些突变对细胞适应性的潜在影响。

  9. 环境对选择压力的影响：在含有复合体V抑制剂奥利霉素的环境中，突变mtDNA的异质性倾向于增加，而在葡萄糖或半乳糖中则倾向于减少。这显示了环境条件如何决定突变是否对细胞有益。

关键发现

  1. 选择作用而非遗传漂移决定异质性变化： 文中通过多条线索证明了在分裂细胞中观察到的经典异质性转移是由作用于细胞适应性水平的选择驱动的，而非简单的遗传漂移或细胞内效应。特别是在标准培养条件下，细胞群体会自动清除对细胞适应性有害的非同义mtDNA变异，而保留无害的同义变异。
  2. 环境因素对异质性动态的重要影响： 研究表明，特定的非同义mtDNA异质性对细胞适应性的影响可以是有害的、中性的或甚至有益的，而这种效应的“符号”完全取决于环境条件。例如，在半乳糖环境中，分裂最频繁的细胞展示最低的非同义异质性；相反，在含奥利霉素的环境中，分裂最频繁的细胞展示最高的非同义异质性。
  3. 异质性在细胞谱系中的稳定性： 通过谱系追踪分析，文中发现尽管整个细胞群体的异质性经历动态变化，但在单个细胞谱系中的异质性却保持稳定。这一发现表明，选择作用主要在细胞群体层面，而非细胞内单个线粒体水平上进行。

局限性

  1. 实验模型的局限：

    • 研究主要依赖于体外细胞培养模型，这可能不完全反映体内情况。细胞培养条件和体内环境存在显著差异，如氧气浓度、营养供应和细胞间相互作用，这些差异可能影响研究结果的适用性和推广性。

  2. 遗传背景的单一性：

    • 研究中使用的细胞线主要是特定类型的人类细胞系，如293T细胞和Nthy-ori 3-1细胞。这些细胞系的遗传背景可能不完全代表更广泛的人群或其他生物种类，限制了发现的普遍性。

  3. 编辑工具的特异性和效率：

    • 使用的基因编辑工具（如DdCBE）虽然能够引入特定的mtDNA突变，但其编辑效率、特异性和潜在的非靶效应未在文中详细讨论。这可能影响实验结果的准确性和解释。

  4. 环境条件的简化模拟：

    • 研究中模拟的环境条件（如使用半乳糖或奥利霉素处理）可能过于简化，无法完全模拟复杂的体内环境，特别是多因素交互作用的情况。

  5. 长期效应和机制研究的不足：

    • 文中主要关注短期内环境变化对异质性动态的影响，缺乏对长期影响和深层机制（如细胞命运决定和遗传稳定性）的系统研究。

  6. 统计和模型验证的局限：

    • 虽然使用了计算模型来解释实验数据，但这些模型的建立和验证可能存在假设的局限性。模型的假设需要在更广泛的生物学背景和实验条件下进行验证和调整。
