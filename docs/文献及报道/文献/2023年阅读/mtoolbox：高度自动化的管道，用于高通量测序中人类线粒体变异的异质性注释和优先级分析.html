摘要

MToolBox: a highly automated pipeline for heteroplasmy annotation and prioritization analysis of human mitochondrial variants in high-throughput sequencing

“MToolBox implements an effective computational strategy for mitochondrial genomes assembling and haplogroup assignment also including a prioritization analysis of detected variants.” (Calabrese 等, 2014, p. 1) MToolBox 为线粒体基因组组装和单倍群分配实施了有效的计算策略，还包括对检测到的变异进行优先级分析。

“MToolBox provides a Variant Call Format file featuring, for the first time, allele-specific heteroplasmy and annotation files with prioritized variants.” (Calabrese 等, 2014, p. 1) MToolBox 提供了变体调用格式文件，首次提供了等位基因特异性异质性和具有优先变体的注释文件。

“MToolBox package is available at https://sourceforge.net/projects/mtoolbox/.” (Calabrese 等, 2014, p. 1)

介绍

“The MToolBox workflow includes a computational strategy to assemble mitochondrial genomes from whole-exome sequencing (WXS) and/or whole-genome sequencing (WGS) data” (Calabrese 等, 2014, p. 1) MToolBox 工作流程包括根据全外显子组测序 (WXS) 和/或全基因组测序 (WGS) 数据组装线粒体基因组的计算策略。“which was further updated to detect insertions and deletions (ins/dels) and to assess the heteroplasmic fraction (HF) of each variant allele with the related confidence interval (CI),” (Calabrese 等, 2014, p. 1) 进一步更新以检测插入和缺失 (ins/dels)，并使用相关置信区间 (CI) 评估每个变异等位基因的异质分数 (HF)，“reported as sample-specific meta-information in an enhanced version of the Variant Call Format (VCF) file (version 4.0).” (Calabrese 等, 2014, p. 1) 在增强版本的变体调用格式 (VCF) 文件（版本 4.0）中报告为特定于样本的元信息。“The MToolBox pipeline analyzes the reconstructed genomes for haplogroup assignment (Rubino et al., 2012) and variant prioritization.” (Calabrese 等, 2014, p. 1) MToolBox 管道分析重建的基因组以进行单倍群分配（Rubino 等人，2012）和变体优先级排序。

MToolBox工作流程的主要步骤：(a-d)读取映射和NumtS过滤； (e–h) 映射后处理； (i-m) 基因组组装、单倍群预测和变异注释。

功能：

  1. 检测核-线片段（nuclear-mitochondrial DNA segments, NUMTs）
  2. 检测异质性（Heteroplasmy）分数
  3. 组装线粒体基因组
  4. 单倍群分配

讨论

“MToolBox is the only tool that generates as output a VCF file, the standard format for large-scale genotyping information, suitably customized for mitochondrial data, by including the heteroplasmy fraction and its related CI.” (Calabrese 等, 2014, p. 2) MToolBox 是唯一生成 VCF 文件作为输出的工具，VCF 文件是大规模基因分型信息的标准格式，通过包含异质性分数及其相关 CI，针对线粒体数据进行了适当定制。
