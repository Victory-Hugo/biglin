分类

在分子生物学和遗传学中，人类参考泛基因组（Human Pangenome Reference ）近年来的重要性逐渐突出。

泛基因组（pangenome）指的是一个进化之上所有的菌株的全部基因的集合。

根据基因在所有菌株基因组中的占比，泛基因组大体上可分为核心基因组（core pangenome）、附属基因组（accessory genome、dispensable genome）、独特基因（singleton genes）。三者之间并无严格的界限，例如，有文献这样划分：core genes是观察到超过99%的核心基因组，soft core genes则是95-99%，shell处于15-95%之间，cloud不足15%。其中shell可认为是前述的附属基因，cloud可认为是独特基因。


根据基因组是否开放，可分为开放泛基因组（open pangenomes）和闭合泛基因组（closed pangenomes），对于闭合泛基因组，其整个泛基因组的大小是可以预测的，而开放泛基因组不行。

可以粗略地通过Heap定律N = kn-α，其中N基因的数量，n表示基因组的数量，k是比例常数，α则决定了泛基因组是否开放，只有小于或等于1时才是开放的。

历史

最早的关于泛基因组的概念是Tettelin等人在研究六株B型链球菌的菌株时提出的，Tettlin等人测序了六株Streptococcus agalactiae的基因序列，它们代表了五种造成疾病的血清型，分析这些基因组和数据库中的序列表明S. agalactiae 物种可以使用泛基因组来描述。于是率先提出，泛基因组由核心基因和附加基因组成，其中核心基因包含了80%以上的菌株所共有的基因，而附属基因只是特定菌株含有的基因。

后来随着测序技术的发展，大量的基因组数据产生了（到2020年，已有产生了超过30000个完整的基因组草图），泛基因组这个领域变得越来越火热。在2015年到2020年五年间，在PubMud上以“pangenome” 为关键词的文章就有近500篇。

与此同时，各种各样的泛基因组分析工具也被开发了出来，它们大都具有相同的功能，即聚类直系同源基因、识别单核苷酸多态性、构建系统发育树以及最后生成泛基因组图。比较旧的工具有：PGAP、PanGP、Micropan、GET_HOMOLOGUES等，近几年新开发的工具有Panaroo、Roary、PanACoTA等。

分析方法

通常来说，泛基因组分析的主要步骤包括对基因组注释信息进行同质化，随后根据直系基因进行聚类，最后作出泛基因组图，这三个主要的步骤每一个都可以通过不同的方法来实现。

基因注释的同质化通常可以使用RAST和Prokka来执行，其目的是避免将核心基因错误分类为附属基因、附属基因错误分类为独特基因。

聚类分析通常通过首先执行全对全双向（all-vs-all bidirectional ）BLAST 分析，然后使用直系同源识别软件（如OrthoMCL、Orthofinder）来实现，也可以通过使用 BLAST 的结果直接根据比对的大小和序列比对的同一性来定义直系同源性，方法是为两者设置一个阈值。

作用

一般而言，泛基因组旨在捕获一个物种内的完整遗传多样性，并减少使用单一参考基因组所固有的遗传分析偏差。在细菌中，泛基因组的分析可以用来研究基因组多样性和系统发育、疾病爆发 、毒力相关基因和抗菌素耐药性。
